## CodeProbe self-check before submission

This project includes a local CodeProbe kit in the `codeprobe/` folder. Launch the tool from `codeprobe/src/`.

Before submission, run CodeProbe on the code **you authored for this project**. Do **not** include starter code, third-party libraries, generated files, minified assets or documentation.

The reported AI-assistance suspicion score is a **formative signal**, not proof of misconduct. Our working expectation is that student-authored code should normally remain **below 60%**. If your result is above **60%**, revise the code, run the check again and complete the disclosure note if the module requires it.

Where manual review is needed, academic judgement will be based on the CodeProbe report together with Git history, intermediate commits, design notes and your ability to explain the submitted code.
